# Inspiration & Design References

This folder contains visual references, design inspiration, and mockups for the project.

## Organization

Feel free to organize images by:
- Feature (e.g., `games/`, `dashboard/`, `header/`)
- Type (e.g., `colors/`, `layouts/`, `components/`)
- Source (e.g., `dribbble/`, `behance/`, `screenshots/`)

## Usage

- Store screenshots of design inspiration
- Keep reference images for UI/UX decisions
- Save color palette references
- Store mockups and wireframes

**Note:** This folder is not tracked in git by default. Add specific files to git if you want to share them with the team.
